# shmtu-dotnet-lib

感谢您使用shmtu-dotnet-lib包！

此包提供了上海海事大学的.NET类库，提供了一些常用的工具和功能。

安装说明：
1. 打开包管理器控制台（PMC）。
2. 运行`Install-Package shmtu-dotnet-lib -Version 1.0.0`。
